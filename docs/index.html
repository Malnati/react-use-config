React Use Config

A simple and flexible hook for managing configuration parameters in React projects.

What is React Use Config?

React Use Config provides the useConfigParam hook, which retrieves configuration values based on the following priority:

  1. Query Parameters: From the URL.
  2. Environment Variables: Prefixed with REACT_APP_.
  3. Default Values: A fallback if the other options are not set.

Features

  • Retrieve parameters dynamically from the URL or environment variables.
  • Supports boolean conversion for parameters like true and false.
  • Customizable default values for unmatched configurations.
  • Lightweight and easy to integrate into any React project.

Getting Started

To start using React Use Config, follow the steps below:

  1. Install the package:
    2. npm install react-use-config
  2. Use the hook in your project:
    3. import useConfigParam from 'react-use-config';
  3. Example:
    4. 
const geoServer = useConfigParam('geoServer', 'http://localhost:8080/geoserver');
console.log(geoServer);
