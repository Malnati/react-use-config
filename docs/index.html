Welcome to React Use Config

This is the GitHub Pages site for the react-use-config project.

The useConfigParam hook helps you manage configuration parameters efficiently by prioritizing:

  • Query Parameters
  • Environment Variables
  • Default Values

Getting Started

Check the GitHub Repository for detailed instructions on installation, usage, and examples.

Key Features

  • Easy integration into React projects.
  • Supports dynamic configuration from multiple sources.
  • Compatible with TypeScript.
